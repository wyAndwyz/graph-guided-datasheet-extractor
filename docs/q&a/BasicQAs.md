# What is GraphRAG?
Opposite to baseline RAG, it uses KGs instead of classic real-world information sources, such as text with vector-similarities, to improve the answer quality especially by komplex reasoning tasks.
All RAG methods benefit the LLM-performance with private information data set.
The baseline RAG cannot offer the semantic connetions between the information pieces.
*source: https://microsoft.github.io/graphrag/*
